# User Registration Service

## Purpose
This project provides a user registration service that allows users to create and manage their accounts in a secure and efficient manner.

## Tech Stack
- **Java**: The programming language used for developing the application.
- **Spring Boot**: A framework that simplifies the setup and development of new Spring applications.
- **Maven**: A build automation tool used for managing the project's dependencies and building the application.

## How to Run
1. Make sure you have [Java](https://www.oracle.com/java/technologies/javase/javase-jdk8-downloads.html) installed (version 8 or higher).
2. Clone the repository:
   ```bash
   git clone https://github.com/arti211/UserRegistrationService.git
   cd UserRegistrationService
   ```
3. Build the project using Maven:
   ```bash
   mvn clean install
   ```
4. Run the application:
   ```bash
   mvn spring-boot:run
   ```
5. Access the application at `http://localhost:8080`.
